\doxysubsection{Detailed Description}
Implements image storage medium. Might be for example an Open\+GL texture or a memory region containing I420-\/data.

Reference counting is needed since a video buffer can be shared between multiple Video\+Sinks, and the buffer needs to be returned to the Video\+Source as soon as all references are gone.
