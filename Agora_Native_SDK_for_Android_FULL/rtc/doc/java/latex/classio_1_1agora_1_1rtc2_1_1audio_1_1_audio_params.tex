\doxysubsection{Detailed Description}
The {\ttfamily \mbox{\hyperlink{classio_1_1agora_1_1rtc2_1_1audio_1_1_audio_params}{Audio\+Params}}} class.

You can pass the {\ttfamily \mbox{\hyperlink{classio_1_1agora_1_1rtc2_1_1audio_1_1_audio_params}{Audio\+Params}}} object in the return value of the following callbacks to set the audio data format for the corresponding callbacks\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{interfaceio_1_1agora_1_1rtc2_1_1_i_audio_frame_observer_a9e62f7b9c5cf201cd75ea2f0b2417b43}{get\+Record\+Audio\+Params}}\+: Sets the audio recording format for the \mbox{\hyperlink{}{on\+Record\+Frame}} callback.
\item \mbox{\hyperlink{interfaceio_1_1agora_1_1rtc2_1_1_i_audio_frame_observer_ae9850f39cd04cee8419eb469e9318b8f}{get\+Playback\+Audio\+Params}}\+: Sets the audio playback format for the \mbox{\hyperlink{}{on\+Playback\+Frame}} callback.
\item \mbox{\hyperlink{interfaceio_1_1agora_1_1rtc2_1_1_i_audio_frame_observer_a5fa3838fcba7216a10777e86503f6646}{get\+Mixed\+Audio\+Params}}\+: Sets the audio mixing format for the \mbox{\hyperlink{}{on\+Mixed\+Frame}} callback.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The S\+DK calculates the sample interval according to the {\ttfamily samples\+Per\+Call}, {\ttfamily sample\+Rate}, and {\ttfamily channel\+Cnt} values in the {\ttfamily \mbox{\hyperlink{classio_1_1agora_1_1rtc2_1_1audio_1_1_audio_params}{Audio\+Params}}} object and triggers the following callbacks at the calculated sample interval\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{}{on\+Record\+Frame}}
\item \mbox{\hyperlink{}{on\+Playback\+Frame}}
\item \mbox{\hyperlink{}{on\+Mixed\+Frame}}
\end{DoxyItemize}
\end{DoxyNote}
Sample interval (seconds) = {\ttfamily sample\+Per\+Call}/({\ttfamily sample\+Rate} Ã— {\ttfamily channel\+Cnt}). Ensure that the value of sample interval is equal to or greater than 0.\+01.
