Occurs when the state of the R\+T\+MP streaming changes.

The S\+DK triggers this callback to report the result of the local user calling {\ttfamily add\+Publish\+Stream\+Url} or {\ttfamily remove\+Publish\+Stream\+Url}.

This callback also reports the streaming U\+RL and its current streaming state. When exceptions occur, you can troubleshoot issues by referring to the detailed error description in the {\ttfamily err\+Code} parameter.
