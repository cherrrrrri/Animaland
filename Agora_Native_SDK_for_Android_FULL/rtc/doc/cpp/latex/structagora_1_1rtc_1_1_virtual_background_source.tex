\doxysubsubsection{\texorpdfstring{color}{color}}
{\footnotesize\ttfamily unsigned int agora\+::rtc\+::\+Virtual\+Background\+Source\+::color}

The color of the custom background image. The format is a hexadecimal integer defined by R\+GB, without the \# sign, such as 0x\+F\+F\+B6\+C1 for light pink. The default value is 0x\+F\+F\+F\+F\+FF, which signifies white. The value range is \mbox{[}0x000000,0x\+F\+F\+F\+F\+FF\mbox{]}. If the value is invalid, the S\+DK replaces the original background image with a white background image.

\begin{DoxyNote}{Note}
This parameter takes effect only when the type of the custom background image is {\ttfamily B\+A\+C\+K\+G\+R\+O\+U\+N\+D\+\_\+\+C\+O\+L\+OR}.
