Adds a video filter to the video track.

Add a video filter in either of the following ways\+:
\begin{DoxyItemize}
\item Use the \mbox{\hyperlink{classagora_1_1rtc_1_1_i_media_node_factory}{I\+Media\+Node\+Factory}} object to create a built-\/in video filter.
\item Use a custom video filter by implementing the \mbox{\hyperlink{classagora_1_1rtc_1_1_i_video_filter}{I\+Video\+Filter}} class.
\end{DoxyItemize}

To add multiple filters, call this method multiple times. The order of the added filters depends on when the app successfully adds the filter.
