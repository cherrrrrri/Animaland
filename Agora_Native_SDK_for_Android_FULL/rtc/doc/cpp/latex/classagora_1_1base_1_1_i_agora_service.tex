\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classagora_1_1base_1_1_i_agora_service}{I\+Agora\+Service}} class.

{\ttfamily \mbox{\hyperlink{classagora_1_1base_1_1_i_agora_service}{I\+Agora\+Service}}} is the entry point of Agora low-\/level A\+P\+Is. Use this interface to create access points to Agora interfaces, including R\+TC connections and media tracks.

You can create an {\ttfamily \mbox{\hyperlink{classagora_1_1base_1_1_i_agora_service}{I\+Agora\+Service}}} object by calling \mbox{\hyperlink{group__create_agora_service_gadedc5876942db5dd130c45e88ea44126}{create\+Agora\+Service}}.

You can configure the {\ttfamily \mbox{\hyperlink{classagora_1_1base_1_1_i_agora_service}{I\+Agora\+Service}}} object for different user scenarios on the global level by using {\ttfamily \mbox{\hyperlink{structagora_1_1base_1_1_agora_service_configuration}{Agora\+Service\+Configuration}}}.

Sets the path and size of the S\+DK log files.

The S\+DK records all the log data during the S\+DK runtime in two log files, each with a default size of 512 KB. If you set {\ttfamily file\+Size} as 1024 KB, the S\+DK outputs log files with a maximum size of 2 MB. If the total size of the log files exceeds the set value, the new output log overwrites the old output log.

\begin{DoxyNote}{Note}
To ensure that the output log is complete, call this method immediately after calling \mbox{\hyperlink{classagora_1_1base_1_1_i_agora_service_a6f1fb6e792c564dee46903cbb3bc35fc}{initialize}}.

The log level follows the sequence of O\+FF, C\+R\+I\+T\+I\+C\+AL, E\+R\+R\+OR, W\+A\+R\+N\+I\+NG, I\+N\+FO, and D\+E\+B\+UG.
