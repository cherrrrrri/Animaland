\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classagora_1_1rtc_1_1_i_audio_pcm_data_sender}{I\+Audio\+Pcm\+Data\+Sender}} class.

In scenarios involving custom audio source, you can use the {\ttfamily \mbox{\hyperlink{classagora_1_1rtc_1_1_i_audio_pcm_data_sender}{I\+Audio\+Pcm\+Data\+Sender}}} class to send P\+CM audio data directly to the audio track. If the audio track is disabled, the sent audio data is automatically discarded.
