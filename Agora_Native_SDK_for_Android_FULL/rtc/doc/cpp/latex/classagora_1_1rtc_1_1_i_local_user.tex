\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classagora_1_1rtc_1_1_i_local_user}{I\+Local\+User}} class defines the behavior and state of a local user.

Each R\+TC connection has its own local user. Apps can get the local user object by calling \mbox{\hyperlink{classagora_1_1rtc_1_1_i_rtc_connection_a1cf54986a330713047da51c1503ec86c}{I\+Rtc\+Connection\+::get\+Local\+User}}.

Each local user has two user roles\+: broadcaster (publisher and subscriber) and audience (subscriber only). A publisher publishes audio and video tracks, while audience receive them.

Sets the role of the user.

You can call this method either before or after connecting to an Agora channel\+:
\begin{DoxyItemize}
\item Before connecting\+: This method sets the user role as publisher or subscriber (default).
\item After connecting\+: This method allows you to switch the user role between publisher and subscriber. The \mbox{\hyperlink{classagora_1_1rtc_1_1_i_rtc_connection_observer_a25f2c3352be19aef2e56b4a6e78f3e2a}{on\+Change\+Role\+Success}} and \mbox{\hyperlink{classagora_1_1rtc_1_1_i_rtc_connection_observer_a7c4b7ec0e3854211f85e1ac885b05b67}{on\+Change\+Role\+Failure}} callbacks indicate the result of this method call.
\end{DoxyItemize}
