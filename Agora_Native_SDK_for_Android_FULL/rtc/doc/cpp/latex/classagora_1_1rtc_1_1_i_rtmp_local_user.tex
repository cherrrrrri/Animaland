\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classagora_1_1rtc_1_1_i_rtmp_local_user}{I\+Rtmp\+Local\+User}} class defines the behavior and state of the R\+T\+MP user.

Once connected, a R\+T\+MP connection has its own user. Apps can get the user object using I\+Rtmp\+Connection\+::get\+User.
