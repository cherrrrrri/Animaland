\doxysubsection{Detailed Description}
The {\ttfamily \mbox{\hyperlink{classagora_1_1rtc_1_1_i_media_control_packet_sender}{I\+Media\+Control\+Packet\+Sender}}} class.

You can get the {\ttfamily \mbox{\hyperlink{classagora_1_1rtc_1_1_i_media_control_packet_sender}{I\+Media\+Control\+Packet\+Sender}}} class object from a video track or audio track, and then send media control packets directly. The media control packets are R\+T\+C\+P/\+U\+DP packets that contain media control payload. If the track is disabled, the packets will be automatically discarded.
