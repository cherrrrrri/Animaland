Joins a channel with the connection ID.

You can call this method multiple times to join more than one channels at a time.

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item If you are already in a channel, you cannot rejoin it with the same uid.
\item We recommend using different user I\+Ds for different channels.
\item If you want to join the same channel from different devices, ensure that the user I\+Ds in all devices are different.
\item Ensure that the app ID you use to generate the token is the same with the app ID used when creating the Rtc\+Engine instance. 
\end{DoxyItemize}
\end{DoxyNote}
