Reports which users are speaking and the speakers volume.

This callback reports the I\+Ds and volumes of the loudest speakers at the moment in the channel, and whether the local user is speaking. Once enabled, this callback is triggered at the set interval, regardless of whether a user speaks or not.

The S\+DK triggers two independent {\ttfamily on\+Audio\+Volume\+Indication} callbacks at one time, which separately report the volume information of the local user and all the remote speakers. For more information, see the detailed parameter descriptions.

Calling the \mbox{\hyperlink{classagora_1_1rtc_1_1_i_rtc_engine_a08a34d5b85a225390a18467c92a7b6f9}{mute\+Local\+Audio\+Stream}} method affects the S\+DK\textquotesingle{}s behavior.
\begin{DoxyItemize}
\item If the local user calls the {\ttfamily mute\+Local\+Audio\+Stream} method, the S\+DK stops triggering the local user\textquotesingle{}s callback.
\item 20 seconds after a remote speaker calls the {\ttfamily mute\+Local\+Audio\+Stream} method, the remote speakers\textquotesingle{} callback does not include information of this remote user; 20 seconds after all remote users call the {\ttfamily mute\+Local\+Audio\+Stream} method, the S\+DK stops triggering the remote speakers\textquotesingle{} callback.
\end{DoxyItemize}

\begin{DoxyItemize}
\item {\ttfamily uid} = 0,
\item {\ttfamily volume} = {\ttfamily total\+Volume}, which reports the sum of the voice volume and audio-\/mixing volume of the local user.
\end{DoxyItemize}
\item In the remote users\textquotesingle{} callback, this array contains the following members\+:
\begin{DoxyItemize}
\item {\ttfamily uid} of each remote speaker.
\item {\ttfamily volume}, which reports the sum of the voice volume and audio-\/mixing volume of each remote speaker.
\end{DoxyItemize}

\end{DoxyItemize}\\

{\em speaker\+Number} & The total number of speakers. The value range is \mbox{[}0, 3\mbox{]}.

\item In the remote speakers' callback, the callback reports the I\+Ds and volumes of the three loudest speakers when there are more than three remote users in the channel, and {\ttfamily speaker\+Number} = 3.
