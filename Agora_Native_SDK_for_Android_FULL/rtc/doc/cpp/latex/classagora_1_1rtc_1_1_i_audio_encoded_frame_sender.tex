\doxysubsection{Detailed Description}
The {\ttfamily \mbox{\hyperlink{classagora_1_1rtc_1_1_i_audio_encoded_frame_sender}{I\+Audio\+Encoded\+Frame\+Sender}}} class.

In scenarios involving custom audio source, you can use the {\ttfamily \mbox{\hyperlink{classagora_1_1rtc_1_1_i_audio_encoded_frame_sender}{I\+Audio\+Encoded\+Frame\+Sender}}} class to send encoded audio data directly to the audio track. If the track is disabled, the sent audio data will be automatically discarded.
